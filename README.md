# UK Artwork Map

This project reads a CSV of site locations (DMS or decimal coordinates), converts coordinates to decimal degrees, filters points to a UK map extent, and plots labelled markers on a map image.

Key behaviors:
- Parses coordinates using [`plot_locations.dms_to_dd`](plot_locations.py).
- Loads rows from the CSV at [siteLocationCsvFile/locations.csv](siteLocationCsvFile/locations.csv) using [`plot_locations.get_locations_from_csv`](plot_locations.py).
- Filters out-of-UK points with [`plot_locations.filterNonUKLocations`](plot_locations.py).
- Assigns label edges and orders via [`plot_locations.assign_labels_custom`](plot_locations.py) and computes label positions with [`plot_locations.get_evenly_spaced_positions`](plot_locations.py).
- Renders the map and saves an image using [`plot_locations.plot_uk_map`](plot_locations.py).
- Entry point is [`plot_locations.main`](plot_locations.py).

Prerequisites
- Python 3.x
- Recommended: run the provided setup script to install dependencies.

Quick start
1. Install dependencies:
   ./setup.sh
   (uses [setup.sh](setup.sh))

2. Generate the map:
   ./generate.sh
   (runs [generate.sh](generate.sh), which calls [plot_locations.py](plot_locations.py))

Files
- [plot_locations.py](plot_locations.py) — main script and functions: [`plot_locations.dms_to_dd`](plot_locations.py), [`plot_locations.get_locations_from_csv`](plot_locations.py), [`plot_locations.filterNonUKLocations`](plot_locations.py), [`plot_locations.assign_labels_custom`](plot_locations.py), [`plot_locations.get_evenly_spaced_positions`](plot_locations.py), [`plot_locations.plot_uk_map`](plot_locations.py), [`plot_locations.main`](plot_locations.py)
- [siteLocationCsvFile/locations.csv](siteLocationCsvFile/locations.csv) — input data (CSV)
- [generate.sh](generate.sh) — convenience runner
- [setup.sh](setup.sh) — installs Python packages
- Output: `uk_artwork_map.png` (generated by the script)

Notes
- The script expects the CSV columns named `INDEX`, `LATITUDE`, and `LONGITUDE`.
- DMS parsing follows the pattern implemented in [`plot_locations.dms_to_dd`](plot_locations.py); malformed DMS rows are skipped with an error message.
- Map extent and styling are defined in [plot_locations.py](plot_locations.py).
